# 📊 Marketing Campaign Analysis Dashboard

## 1️⃣ Project Overview

The **Marketing Campaign Analysis Dashboard** is an interactive Power BI dashboard developed to analyze and monitor the performance of marketing campaigns across multiple campaign types, customer segments, marketing channels, and time periods.

The dashboard provides a comprehensive view of key marketing KPIs such as Revenue, Impressions, Clicks, Conversions, and Return on Investment (ROI). Interactive slicers allow users to filter data dynamically and gain meaningful business insights for better decision-making.

---

## 2️⃣ Objective

The primary objectives of this project are:

- Analyze marketing campaign performance.
- Compare revenue generated by different campaign types.
- Monitor monthly revenue trends.
- Evaluate customer segment performance.
- Measure campaign ROI.
- Compare acquisition cost with generated revenue.
- Build an interactive dashboard for business users.
- Enable data-driven marketing decisions.

---

## 3️⃣ Dataset Information

| Column Name | Data Type | Description |
|--------------|-----------|-------------|
| Campaign_ID | Text | Unique identifier for each marketing campaign |
| Campaign_Type | Text | Type of campaign (Email, SEO, Paid Ads, Influencer, Social Media) |
| Target_Audience | Text | Target audience category |
| Duration | Whole Number | Campaign duration in days |
| Channel_Used | Text | Marketing channel used for the campaign |
| Impressions | Whole Number | Number of campaign impressions |
| Clicks | Whole Number | Total clicks received |
| Leads | Whole Number | Number of leads generated |
| Conversions | Whole Number | Number of successful conversions |
| Revenue | Decimal Number | Revenue generated from the campaign |
| Acquisition_Cost | Decimal Number | Marketing cost incurred |
| ROI | Decimal Number | Return on Investment |
| Language | Text | Campaign language |
| Engagement_Score | Decimal Number | Customer engagement score |
| Customer_Segment | Text | Customer segment targeted |
| Date | Date | Campaign execution date |

## 6️⃣ Business Insights

- Social Media and Paid Ads generated the highest overall revenue.
- Email campaigns delivered competitive ROI with relatively lower revenue.
- Monthly revenue trends show fluctuations, indicating seasonal campaign performance.
- Premium Customers and Working Professionals achieved higher conversions.
- Higher acquisition costs generally resulted in higher revenue generation.
- Marketing channel performance varies across campaign types.
- Interactive slicers enable detailed campaign analysis based on user-selected filters.

---

## 7️⃣ Business Recommendations

- Increase investment in high-performing campaign types with strong ROI.
- Optimize underperforming campaigns by reducing acquisition costs.
- Focus marketing efforts on customer segments with higher conversion rates.
- Improve campaign strategies during low-performing months.
- Continuously monitor ROI before increasing marketing budgets.
- Utilize dashboard slicers to analyze campaign-specific performance and support decision-making.
